AX210 Wi-Fi Fix Guide (TXT)

========================
1) AX210_WIFI_REFRESH
========================
หน้าที่:
รีเฟรชการ์ด Wi-Fi Intel AX210 แบบเร็ว
ใช้เมื่อ:
- เน็ตอืด
- Wi-Fi หลุด
- Ping แกว่ง
- ต่อเน็ตแล้วแปลกๆ

วิธีใช้:
1. คลิกขวาไฟล์ AX210_WIFI_REFRESH
2. เลือก Run as administrator
3. รอให้คำสั่งทำงานจนเสร็จ
4. ระบบจะรีเซ็ต / รีเฟรชอะแดปเตอร์ Wi-Fi
5. ลองเชื่อมต่อ Wi-Fi ใหม่

========================
2) AX210_RESTORE_DEFAULT
========================
หน้าที่:
คืนค่าระบบเครือข่ายและ Wi-Fi กลับค่าเดิม
ใช้เมื่อ:
- ปรับแต่งเยอะแล้วเน็ตมีปัญหา
- อยากล้างค่าที่เคย tweak
- อยากกลับค่าเดิมก่อนเริ่มตั้งใหม่

วิธีใช้:
1. คลิกขวาไฟล์ AX210_RESTORE_DEFAULT
2. เลือก Run as administrator
3. รอให้ระบบคืนค่าการตั้งค่า
4. รีสตาร์ทเครื่อง 1 ครั้ง
5. เข้า Wi-Fi ใหม่

========================
3) AX210_GODMODE_FINAL
========================
หน้าที่:
ปรับแต่ง Intel AX210 แบบแรงสุด / ลดหน่วง / เน้นความนิ่ง
ใช้เมื่อ:
- เล่นเกม
- อยากลด ping / jitter
- อยากให้ Wi-Fi ตอบสนองไวขึ้น
- ใช้หลังจากตั้ง DNS และไดรเวอร์เรียบร้อยแล้ว

วิธีใช้:
1. ปิดโปรแกรมที่ใช้เน็ตหนักๆ ก่อน
2. คลิกขวาไฟล์ AX210_GODMODE_FINAL
3. เลือก Run as administrator
4. รอจนคำสั่งทำงานเสร็จ
5. รีสตาร์ทเครื่อง 1 ครั้ง
6. ทดสอบเน็ต / เข้าเกมเช็ก ping

========================
ลำดับแนะนำในการใช้
========================
ถ้าจะเริ่มใหม่ทั้งหมด:
1. AX210_RESTORE_DEFAULT
2. รีสตาร์ทเครื่อง
3. AX210_GODMODE_FINAL
4. รีสตาร์ทเครื่อง
5. ถ้าเน็ตแปลกๆ ภายหลัง ใช้ AX210_WIFI_REFRESH

========================
คำเตือน
========================
- ต้อง Run as administrator
- แนะนำสร้าง Restore Point ก่อนใช้
- หลังใช้ AX210_GODMODE_FINAL ควรรีสตาร์ทเครื่อง
- ถ้าเน็ตมีปัญหา ให้ใช้ AX210_RESTORE_DEFAULT เพื่อคืนค่า
