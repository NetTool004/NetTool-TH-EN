=========================================================
         PRO MENU v4 POWERSHELL GODMODE UI
            Desktop-Friendly Network Tool
=========================================================

[ABOUT]
This tool is a desktop-friendly Wi-Fi refresh and network utility menu
made for general daily use, gaming, streaming, and quick troubleshooting.

It is NOT a magic internet booster.
It helps refresh your local network state, reconnect Wi-Fi, and fix
common Windows network issues safely.

=========================================================
[FILES]
1) RUN_NET_TOOL.bat
   -> Launch this file (Recommended)

2) PRO_MENU_V4_GODMODE_UI.ps1
   -> Main PowerShell script

=========================================================
[HOW TO USE]
1) Put all files in the SAME folder
   Example:
   C:\NetTool\

2) Right-click RUN_NET_TOOL.bat
   -> Run as Administrator (Recommended)

3) Use the menu options as needed

=========================================================
[MENU GUIDE]

[1] SAFE DAILY
- Light and safe network refresh
- Recommended for daily use
- Good for general browsing / Discord / YouTube / light gaming

Use when:
- Internet feels slightly weird
- Small lag spikes
- Want a safe refresh before normal use

---------------------------------------------------------

[2] SMART REFRESH
- Stronger refresh than SAFE DAILY
- Good before gaming or heavy internet use
- Recommended before Roblox / FPS / downloads / streaming

Use when:
- Before gaming
- Before long sessions
- When connection feels inconsistent

---------------------------------------------------------

[3] RECONNECT WIFI
- Temporarily disables and re-enables your Wi-Fi adapter
- Forces a clean reconnect to the router

Use when:
- Wi-Fi connected but feels bugged
- Random packet issues
- Need a quick reconnection

---------------------------------------------------------

[4] FULL CLEAN FIX
- Stronger repair mode
- Includes Winsock reset + Wi-Fi reconnect
- Use only when needed

Use when:
- Internet is clearly broken
- Some apps fail to connect
- DNS / socket / weird Windows network issue

NOTE:
- Not recommended to spam often
- Best used only when necessary

---------------------------------------------------------

[5] SHOW INTERNET DIAG
- Shows internet/network diagnostic info

Use when:
- You want to inspect network status
- Troubleshooting

---------------------------------------------------------

[6] SHOW WIFI DETAILS
- Shows detailed Wi-Fi information
- Signal, channel, radio type, etc.

Use when:
- Checking Wi-Fi quality
- Checking if connected to 2.4GHz or 5GHz

---------------------------------------------------------

[7] QUICK PING TEST
- Quick latency test to check responsiveness

Use when:
- You want to check if the internet feels delayed
- Testing before gaming

---------------------------------------------------------

[8] SHOW IP CONFIG
- Displays IP address, gateway, DNS, adapter info

Use when:
- You want to verify current IP / DNS
- Troubleshooting internet settings

=========================================================
[RECOMMENDED USAGE]

Daily normal use:
- Option [1] SAFE DAILY

Before gaming / streaming / heavy use:
- Option [2] SMART REFRESH

If Wi-Fi feels stuck:
- Option [3] RECONNECT WIFI

If Windows network is acting broken:
- Option [4] FULL CLEAN FIX

=========================================================
[IMPORTANT NOTES]

- This tool does NOT increase your ISP speed beyond your internet plan
- It helps clean, refresh, and stabilize local Windows network behavior
- Best results depend on:
  * Router quality
  * Distance from router
  * Wi-Fi signal strength
  * ISP stability
  * Network congestion

=========================================================
[RECOMMENDED SETUP]

Best folder:
C:\NetTool\

Avoid:
- Very long folder paths
- Special character heavy folders
- Cloud sync folders (sometimes)
- OneDrive paths (can work, but C:\NetTool\ is cleaner)

=========================================================
[TROUBLESHOOTING]

If the script closes too fast:
1) Run RUN_NET_TOOL.bat as Administrator
2) Make sure PowerShell is available
3) Move files to:
   C:\NetTool\
4) If blocked, allow PowerShell scripts temporarily:
   PowerShell (Admin):
   Set-ExecutionPolicy -Scope Process Bypass
