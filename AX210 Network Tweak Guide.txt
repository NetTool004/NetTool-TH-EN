AX210 Network Tweak Guide (TXT)

========================
1) Net_Tweak_FULL_AX210_AUTO
========================
หน้าที่:
ปรับแต่งระบบเครือข่ายแบบอัตโนมัติสำหรับ Intel AX210
เน้น:
- ลด Ping
- ลด Delay
- ลด Jitter
- เพิ่มความนิ่งของ Wi-Fi
- ปรับค่าระบบเน็ตให้เหมาะกับการเล่นเกม

เหมาะสำหรับ:
- เล่นเกมออนไลน์
- ใช้ Wi-Fi AX210
- เน็ตมีอาการหน่วง / ไม่เสถียร
- อยากให้ระบบปรับให้แบบอัตโนมัติ

วิธีใช้:
1. ปิดโปรแกรมที่ใช้เน็ตหนักๆ ก่อน
2. คลิกขวาไฟล์ Net_Tweak_FULL_AX210_AUTO
3. เลือก Run as administrator
4. รอให้คำสั่งทำงานจนเสร็จ
5. รีสตาร์ทเครื่อง 1 ครั้ง
6. เข้าเกม / ทดสอบ Ping ใหม่

========================
2) AX210_Gaming_Tweak
========================
หน้าที่:
ปรับแต่ง Intel AX210 สำหรับการเล่นเกมโดยเฉพาะ
เน้น:
- ตอบสนองไวขึ้น
- ลดอาการกระตุกจาก Wi-Fi
- ช่วยให้การเชื่อมต่อเสถียรขึ้น
- เหมาะกับเกม FPS / เกมที่ต้องการ ping ต่ำ

เหมาะสำหรับ:
- ใช้ AX210 เล่นเกม
- เน้นความลื่นและความนิ่ง
- ใช้คู่กับ DNS ที่ตั้งแล้ว

วิธีใช้:
1. เชื่อมต่อ Wi-Fi ให้เรียบร้อยก่อน
2. คลิกขวาไฟล์ AX210_Gaming_Tweak
3. เลือก Run as administrator
4. รอให้คำสั่งทำงานเสร็จ
5. รีสตาร์ทเครื่อง 1 ครั้ง
6. ทดสอบเกม / เช็กค่า Ping

========================
ลำดับแนะนำในการใช้
========================
ถ้าจะใช้ทั้ง 2 ไฟล์:
1. Net_Tweak_FULL_AX210_AUTO
2. รีสตาร์ทเครื่อง
3. AX210_Gaming_Tweak
4. รีสตาร์ทเครื่อง
5. ทดสอบเน็ต / เข้าเกมเช็ก ping

========================
คำแนะนำเพิ่มเติม
========================
- ควรตั้ง DNS ก่อนใช้งาน
- ควรอัปเดตไดรเวอร์ Intel AX210 ให้ใหม่
- ถ้าใช้แล้วมีปัญหา ให้ใช้ไฟล์ Restore Default (ถ้ามี)
- ต้อง Run as administrator ทุกครั้ง
